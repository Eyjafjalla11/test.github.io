    1. 比赛（Match）&决斗准备
      1. 必要装备
        1. 卡组（主卡组）
          • 由怪兽卡（通常怪兽+效果怪兽+仪式怪兽+灵摆怪兽）+魔法卡+陷阱卡组成。
          • 卡片数目必须在40～60张之间。
        2. 额外卡组
          • 由额外怪兽组成（同调怪兽+超量怪兽+融合怪兽+连接怪兽/Link怪兽）。
          • 卡牌数目应当在0～15张之间。
        3. 副卡组
          • 由怪兽卡（全部的怪兽）+魔法卡+陷阱卡组成。
          • 卡牌数目应当在0～15张之间。
          • 在三局两胜比赛中的两局决斗之间，可以与主卡组或额外卡组的卡牌进行一对一替换。
          • 卡组构建限制:
            • 卡名相同的卡牌在卡组·额外卡组·副卡组中，合计最多放入3张。
            • 卡牌构筑需要参考「禁止·限制·准限制卡表」（官方比赛必须遵守）。
      2. 某些场合的必要装备：
          硬币（正反两面需要有明显区别）
          • 需要在卡牌的效果需要硬币进行判定时进行使用。
           骰子（六个面需要有1～6的数字，并且质地均匀）
          • 需要在卡牌的效果需要骰子进行判定时进行使用。
           指示物 怪兽衍生物（Token）
      3. 方便决斗而非必要装备：
        1. 计算器
          • 方便进行伤害计算。
        2. 卡套
          • 可以保护卡牌不被磨损。
      4. 决斗场地
        • 一共分为7个区域：怪兽区（额外怪兽区+主要怪兽区）、魔法·陷阱区、场地区、墓地区、除外区、主要卡组区、额外卡组区，共32格（2格公共格+双方各15格）。
        1. 怪兽区
          1. EX怪兽区（额外怪兽区）
            • 共2格：用于放置从额外卡组中召唤的怪兽——融合怪兽、同步怪兽、超量怪兽、灵摆怪兽、和连接怪兽（一般为连接召唤，连接怪兽的所对应箭头的区域并不能算作EX区）。
            • 通常情况下，一方玩家只能使用一格（可以通过类似U型锁的方式来获得另一格额外怪兽格，获得的额外怪兽格上的怪兽不算作在自己场上）。
          2. 主要怪兽区
            • 共5格：用于放置从非额外卡组召唤的怪兽（通常怪兽+效果怪兽）+从额外卡组召唤的怪兽。
            • 左到右分别是一至五号。
            • 位于额外怪兽区下方，其中二号位和四号位上方分别对应两格额外怪兽格。
        2. 魔法·陷阱区
          • 共5格：用于放置魔法卡（场地魔法卡除外）+陷阱卡+作为魔法卡和灵摆刻度的灵摆怪兽。
          • 位于主要怪兽区下方。
          • 灵摆区
            • 共2格：魔法·陷阱区最左边和最右边
            • 可以作为灵摆区放置灵摆刻度（灵摆怪兽）。
        3. 场地区
          • 共1格：用于放置场地魔法卡。
          • 位于主要怪兽区的左侧。
        4. 墓地区
          • 共1格：用于放置非
          • 正常离场（“除外”除外）的怪兽卡+魔法卡+陷阱卡。
          • 位于主要怪兽区的右侧。
        5. 除外区
          • 共1格：用于放置被除外的怪兽卡+魔法卡+陷阱卡。
          • 位于墓地区上方。
        6. 主要卡组区
          • 共1格：用于放置主要卡组。
          • 位于魔法·陷阱区右侧。
        7. 额外卡组区
          • 共1个：用于放置额外卡组。
          • 位于魔法·陷阱区左侧。
    2. 卡牌种类介绍
      • 一共分为3种：怪兽卡、魔法卡、陷阱卡。
      1. 怪兽卡
        • 一共分为8种类型：通常怪兽、效果怪兽、仪式怪兽、融合怪兽、同调怪兽、超量怪兽、灵摆怪兽、连接怪兽（Link怪兽）。
        • 通常情况下，一方玩家可以召唤6只怪兽。
        • 怪兽卡通用信息：
        • 卡名：表示怪兽的名称。在规则上，是判断卡牌是否为同一张卡牌的唯一标准。
        • 字段：某些怪兽名字中的某些特定前缀（如同怪兽卡的姓氏）。
        • 属性（共6种）：地、水、炎、风、光、暗。
        • 种族（共25种）：魔法师族、龙族、幻龙族、恐龙族、海龙族、不死族、战士族、兽战士族、兽族、鸟兽族、机械族、恶魔族、天使族、昆虫族、爬虫族、鱼族、水族、炎族、雷族、岩石族、植物族、念动力族、电子界族、幻神兽族、创造神族。
        • 等级：以红圆黄星作为标识，共12星（最低为1星，最高12星）。
        • 文字描述栏：怪兽的介绍（通常是通常怪兽或无特殊效果的怪兽）、怪兽效果、或者特殊的召唤条件。
        • 表现形式：攻击表示（以直立的方向摆放）、守备表示（以横向的方向摆放）、里侧表示（背面朝上）、表侧表示（正面朝上）。
        • 放置位置：主卡组（通常怪兽+效果怪兽+仪式怪兽+灵摆怪兽）、额外卡组（融合怪兽+同调怪兽+超量怪兽+连接怪兽/Link怪兽）。
        • 怪兽卡的通用限制：
          • 通常召唤（通常怪兽+效果怪兽的召唤）：规则上一回合一次，1至4星怪兽无需祭品、5星和6星怪兽需要1只祭品、7星以上怪兽需要2只祭品、某些特殊怪兽需要3只祭品，需要祭品的通常召唤为上级召唤。
          • 特殊召唤（仪式召唤、融合召唤、同调召唤、超量召唤、连接召唤）：规则上，无限制次数。
          • 特殊召唤（灵摆召唤）：规则上，一回合一次。
          • 表现形式：主动的变更攻守表示一回合一次（效果除外）。
          • 苏生限制：当怪兽没有通过正规方式出场或者没有被成功召唤的情况下，怪兽无法被自己卡片文本规定以外的方式从墓地或被除外的状态中进行苏生（重现召唤）。当怪兽被正规出场时，该限制才可能解除。
        1. 通常怪兽
          • 没有任何效果的怪兽，俗称“白板”。
          • 卡牌颜色：黄色。
          • 召唤条件：通常召唤。
        2. 效果怪兽
          • 拥有特殊效果的通常怪兽。
          • 特殊种类：
            •
          • 卡牌颜色：橙色。
          • 召唤条件：通常召唤
            • 反转怪兽：怪兽从表侧表示变更为里侧表示时，才可以发动效果。
            • 卡通怪兽： 拥有“卡通”字段的一系列怪兽，通常具有直接攻击的能力。
            • 灵魂怪兽： 大部分拥有着召唤限制和结束阶段回到手卡的效果。
            • 同盟怪兽： 拥有能成为指定怪兽的装备的能力，也能解除装备回复怪兽状态。
            • 二重怪兽： 在场上和墓地时，被当作通常怪兽。在场上时，可以进行二度召唤而变成效果怪兽获得相关效果。
          • 衍生概念：
            •
            • 反转效果：卡牌从里侧表示变为表侧表示时自动发动效果。
            • 起动效果：自己的回合主动使用的效果。
            • 永续效果： 只要该怪兽表侧存在于场上时，效果则一直存在（当怪兽以表侧表示出现在场上的时点即适用效果，怪兽离场时则效果消失）。
            • 诱发效果：满足某个时点或场合而发动的效果。
            • 诱发即时效果：可以在任意回合（包括对方回合）、并且在满足发动条件的情况下自由发动的效果。
        3. 仪式怪兽
          • 分为没有特殊效果和拥有特殊效果两种怪兽。
          • 卡牌颜色：蓝色。
          • 召唤条件：手中持有仪式怪兽+手中持有特定的仪式魔法卡+满足特定仪式魔法卡的召唤条件 （将若干怪兽作为素材、将若干怪兽除外、将若干怪兽直接送墓、或者直接支付基本分等条件）。
        4. 融合怪兽
          • 分为没有特殊效果和拥有特殊效果两种怪兽。
          • 卡牌颜色：紫色。
          • 召唤条件：通常情况下，在场上拥有足够的特定素材+手中持有融合魔法卡。
        5. 同调怪兽
          • 拥有特殊效果。
          • 卡牌颜色：白色。
          • 召唤条件：通常情况下，调整怪兽（通常为1只）+非调整怪兽若干作为素材+作为素材的怪兽的等级与所召唤的同调怪兽的等级必须相等。
          • 衍生概念：
            • 调整怪兽
              • 与效果怪兽的概念不同（绝大多数调整怪兽与效果怪兽重叠）。
              • 是作为召唤同调怪兽的必要条件。
        6. 超量怪兽
          • 拥有特殊效果，需要以超量素材作为效果发动的代价。
          • 卡牌颜色：黑色。
          • 卡牌阶级（而非等级）：以黑圆黄星为标识，阶级为1至12星。
          • 召唤条件：将若干相同等级的怪兽（等级必需与所召唤的超量怪兽的阶级相同） 作为超量素材进行叠放
          • 衍生概念：
            • 是超量怪兽发动效果的代价。
            • 作为素材的怪兽卡·魔法卡·陷阱卡在场上不视为卡片。
            • 被送去墓地的场合，不视为卡片从场上送去墓地。
            • 通常情况下，不会成为效果对象。
        7. 灵摆怪兽
          • 怪兽卡&魔法卡的综合体——在灵摆区域可以当作「魔法卡」。
          • 种类分为效果灵摆怪兽、仪式灵摆怪兽、融合灵摆怪兽、同调灵摆怪兽、 超量灵摆怪兽。
          • 卡牌颜色：下半部分为绿色（魔法卡）。
          • 卡牌的文字描述：下半部分是作为怪兽卡的描述，上半部分是作为魔法卡的描述。
          • 召唤方式：通常召唤+特殊召唤+灵摆召唤。
          • 送墓规则：
            • 当灵摆怪兽需要从场上被送去墓地时，灵摆怪兽不会被送去墓地，而是表侧表示加入额外卡组。
            • 当灵摆怪兽需要从场上以外的地方被送去墓地时，灵摆怪兽不会加入额外卡组，而是需要送去墓地。
            • 当被当作超量素材使用的灵摆怪兽被取除时，灵摆怪兽需要被送去墓地。
            • 场上的灵摆怪兽不能作为送墓地的代价（COST）使用。

      2. 魔法卡
        • 一共分为6种类型：普通魔法（无符号）、装备魔法（十字符号）、场地魔法（手里剑符号）、永续魔法（无穷大符号）、速攻魔法（闪电符号）、仪式魔法（火焰符号）。
        • 卡牌颜色：绿色。
        • 文字描述栏：卡牌效果。
        • 发动条件：除速攻魔法卡，只能在自己的主要阶段发动，并且需要占用1格魔法·陷阱区。
      3. 陷阱卡
        • 一共分为三个类型：通常陷阱（无符号）、永续陷阱（无穷大符号）、反制陷阱（箭头）。
        • 卡牌颜色：红色。
        • 文字描述栏：卡牌效果。
        • 发动条件：通常情况下，陷阱卡必须先被设置（SET）于魔法·陷阱卡区，并且在当前回合结束后才可发动。在此之后,可以在任意回合发动。
    3. 比赛（Match）&决斗流程
      • 一场比赛（Match）中最多有3局决斗，总限制时间为40分钟（每场Match实行三局两胜制）。
      • Match胜利条件：
      • 先胜2局决斗的者，此参赛者胜利。
      • 1胜2平，获得1胜者胜利。
      • 1胜1平1负或3平的场合，双方平局。

    4. 战斗阶段：
      1. 开始步骤：
        • 记录为「战斗阶段开始时」的卡牌效果在此时可以发动。
      2. 战斗步骤：
        • 先指定自己的怪兽，再指定对手的怪兽。
        • 在此阶段，可以发动记录为「攻击宣言时」和「成为攻击对象时」等的卡牌效果。
      3. 伤害计算步骤：
        • 计算怪物战斗造成的伤害。
        • 一共分为5个阶段：伤害步骤开始时、伤害计算前、伤害计算时、伤害计算后、伤害步骤结束时。
      4. 结束步骤：
        • 宣布战斗阶段结束。
        • 在此阶段可发动记录为「战斗阶段结束时」的卡牌效果。
    5. 卡牌速度&连锁
      • 卡牌效果的发动速度。
      • 咒文速度1（通常魔法、装备魔法、场地魔法、仪式魔法、永续魔法、反转效果、效果怪兽的起动效果和诱发效果）：速度最慢，不能自发进入连锁。
      • 咒文速度2（速攻魔法、通常陷阱、永续陷阱、效果怪兽的诱发即时效果）：速度中等，可以连锁咒文速度1和2的效果。
      • 咒文速度3（反制陷阱）：速度最快，可以连锁咒文速度1、2和3的效果。
    6. 其他规则
      • 卡牌的送墓顺序：如果有多张卡牌被同时送墓时，左侧的卡牌优先送墓。
      • 「猜谜」规则：根据魔法卡「猜谜」效果，墓地中的卡牌的顺序不可以更改
